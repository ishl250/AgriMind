# 🌱 AgriMind AI

> Empowering farmers with AI-powered crop disease detection, soil analysis, treatment recommendations, and weather forecasting.

## 🏆 Achievement

AgriMind AI was developed during the **COODIC x MLH Gemini Hack Kigali Hackathon** and was recognized as one of the winning projects. The platform leverages Artificial Intelligence to help farmers make smarter, data-driven decisions.

---

## 📖 Overview

Agriculture faces significant challenges such as crop diseases, poor soil conditions, and unpredictable weather. AgriMind AI addresses these challenges by providing farmers with intelligent recommendations and real-time insights.

The platform uses AI and machine learning technologies to analyze crop images, recommend treatments, evaluate soil conditions, and provide weather forecasts.

---

## ✨ Features

### 🔍 Crop Disease Detection

* Upload crop images.
* AI-powered disease identification.
* Confidence score and diagnosis results.

### 💊 Treatment Recommendations

* Disease-specific treatment suggestions.
* Prevention and management tips.

### 🌱 Soil Analysis

* Analyze soil characteristics.
* Recommend suitable crops.
* Improve crop productivity.

### 🌦 Weather Forecasting

* Real-time weather updates.
* Short-term forecasts.
* Agricultural planning support.

### 🤖 AI Farming Assistant

* Smart farming recommendations.
* Crop management guidance.
* Decision support system.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/ishl250/AgriMind.git
cd agrimind-ai
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
npm run dev
```

### Install Backend Dependencies

```bash
cd backend
npm install
npm start
```

### Environment Variables

Create a `.env` file:

```env
PORT=5000

MONGO_URI=your_database_url

GEMINI_API_KEY=your_gemini_api_key

WEATHER_API_KEY=your_weather_api_key
```
